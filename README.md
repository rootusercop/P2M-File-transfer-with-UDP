Point-to-Multipoint File Transfer Protocol with UDP [Oct’13]
Created multithreaded Server and Client in Java to exchange data over the internet using UDP
Implemented a stop-and-wait protocol for communication between Client and Server. Programmed Socket Interfaces
Synchronized all the Server threads with each other for simultaneous synchronization with the Client for receiving data packets
Emulated a loss condition to loose packets sent by the Sender to the Receiver and ACK from the Receiver to the Sender
